# Content Strategy Reference

When to use: Writing comparison copy, structuring alternative suggestions, or creating SEO-friendly comparison content for IDX plan tiers.

## Patterns

### Tier-Based Positioning

Align feature descriptions with buyer personas:

| Tier | Positioning | Key Phrase |
|------|-------------|------------|
| Pro ($39) | "Solo agents getting started" | "Essential MLS access" |
| Smart ($79) | "Growing teams" | "Lead capture + GHL sync" |
| Ultra ($179) | "Power users" | "Unlimited domains + API keys" |
| Mega ($449) | "Brokerages" | "Custom branding + SLA" |

### Alternative Page Headlines

Structure H1s for programmatic comparison pages:

```
"{Current Plan} vs {Alternative Plan}: Which IDX Tier is Right for You?"
"Best {Category} Alternative to {Competitor} for Tampa Bay Agents"
"Compare {Feature}: Pro vs Smart vs Ultra"
```

### Feature Differentiation Copy

Avoid generic feature lists. Quantify where possible:

- ❌ "Lots of API calls"
- ✅ "2M API calls/month (Ultra) vs 100K (Smart)"

- ❌ "MLS data included"
- ✅ "Stellar MLS real-time sync with 15-min cache"

## Pitfalls

Never claim "unlimited MLS data" in comparison copy—even Mega tier has metered overage billing. Use precise language: "Unlimited domains" and "2M API calls/month" are accurate; "unlimited data" violates MLS compliance and creates billing disputes.
